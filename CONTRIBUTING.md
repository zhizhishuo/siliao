# Contributing / 贡献指南

Thank you for considering contributing! 感谢你考虑为本项目做出贡献！

## Development Setup / 开发环境

```bash
git clone https://github.com/zhizhishuo/siliao.git
cd siliao
npm install
npm run dev
```

Frontend: http://localhost:5173
Backend: http://localhost:3001

## Pull Request Process / PR 流程

1. Fork the repository / Fork 本仓库
2. Create a feature branch / 创建功能分支: `git checkout -b feature/my-feature`
3. Commit your changes / 提交更改: `git commit -m "Add my feature"`
4. Push to the branch / 推送分支: `git push origin feature/my-feature`
5. Open a Pull Request / 发起 PR

## Code Style / 代码规范

- Frontend: ESLint + TypeScript strict mode
- Backend: ES modules, consistent naming
- Run `npm run lint` before submitting
